# MyAvatar

Projet reprenant le système de Gravatar pour le projet globale PoPotion

### Technologies

| php | BDD           |
| --- | ------------- |
| 8.2 | mysql/mariadb |

### Configuration

Configuration à mettre dans un fichier `.env.local`

```
DATABASE_URL=mysql://root:root@127.0.0.1:3307/project?serverVersion=mariadb-10.5.8
```
